## Что подготовить на сайте перед записью

1. **Фермер bereke** — иметь 1 заявку в статусе `submitted` (подана, ещё не проверена)
2. **Новая заявка** — подготовить данные для быстрой подачи (направление, вид, количество)
3. **Специалист** — убедиться что заявка bereke видна в списке
4. **Комиссия** — иметь 1-2 заявки в статусе `checking` для демо одобрения
5. **Очередь выплат** — иметь несколько платежей с разными Merit Score

Для подготовки данных зайти на https://subsidyai.109-235-119-92.sslip.io и пройти flow до нужного этапа.
