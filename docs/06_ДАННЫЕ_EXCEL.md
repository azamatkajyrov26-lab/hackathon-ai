## Что можно извлечь для скоринга

### Из самих данных:
- **Частота подачи заявок** одним хозяйством (по номеру заявки / району)
- **Суммы субсидий** по хозяйствам
- **Распределение по регионам** (области, районы)
- **Распределение по направлениям** животноводства
- **Статусы заявок** (исполнена, отказ, в обработке)
- **Временные паттерны** (когда подают — начало года vs конец)

### Что нужно сгенерировать (обезличено):
- ИИН/БИН (синтетические)
- Номера учётных записей
- Данные о поголовье (количество голов, породы)
- Земельные участки
- ЭСФ
- Встречные обязательства (рост/падение валовой продукции)

## Статусы заявок (возможные)
- Исполнена — субсидия выплачена
- На рассмотрении — в процессе проверки
- Отказано — мотивированный отказ
- В листе ожидания — одобрена, ждёт финансирования
- Частично выплачена — частичная оплата при нехватке средств
