# Карта интеграций — план эмуляции

```
hackathon-ai/
├── config/                    # Основной проект
├── apps/
```

### 7. ИС "Клиент-Казначейство" — Платежи

**Endpoint:** `POST /api/emulator/treasury/submit-payment/`

**Request:**
```json
{
  "payment_id": "PAY-2025-00123",
  "applicant_iin_bin": "123456789012",
  "bank_account": "KZ123456789012345678",
  "amount": 5200000,
  "subsidy_type": "Приобретение маточного поголовья КРС",
  "xml_data": "<payment>...</payment>"
}
```

**Response:**
```json
{
  "status": "accepted",
  "treasury_reference": "TR-2025-98765",
  "payment_date": "2025-03-20",
  "amount_paid": 5200000,
  "form_5_15": {
```

## Источник данных для эмуляции

Основа — Excel-выгрузка (36 658 записей):
- Область, Акимат, Район хозяйства → для ЕГКН, ИАС "РСЖ"
- Направление, Наименование субсидирования → для ИС "ИСЖ", ЕАСУ
- Норматив, Причитающая сумма → для ИС "ЭСФ", Казначейство
- Статус заявки → для истории в ИАС "РСЖ"
- Дата поступления → для хронологии

Дополнительно: генерация синтетических данных для полей, которых нет в выгрузке (ИИН/БИН, номера животных, земельные участки).
