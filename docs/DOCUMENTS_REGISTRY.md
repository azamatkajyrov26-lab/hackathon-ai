# Реестр документов и сертификатов

Полный перечень документов, которые заявитель загружает при подаче заявки, с привязкой к формам и направлениям. Источник: Приложения 3, 4, 5 Правил субсидирования.

---

## Общие документы (обязательны для ВСЕХ форм)

| # | Документ | Формат | Описание | Основание |
|---|----------|--------|----------|-----------|
| 1 | Договор купли-продажи | PDF/JPG | Оригинал или нотариальная копия | Приложение 3, общие поля |
| 2 | Документ, подтверждающий полную оплату | PDF/JPG | Платежное поручение, выписка банка | Приложение 3, общие поля |
| 3 | ЭСФ (электронная счет-фактура) | PDF | Из ИС ЭСФ, подтверждение затрат | п.8 Правил |

## Документы по направлениям

### Форма 1: Приобретение племенного маточного поголовья КРС, овец, баранов-производителей

| # | Документ | Обязательный | Описание |
|---|----------|:------------:|----------|
| 1 | Общие документы (1-3) | Да | См. выше |
| 2 | Племенное свидетельство | Да | На каждое приобретённое животное, выданное племенным центром |
| 3 | Ветеринарный сертификат | Да | Подтверждение здоровья животных |
| 4 | Акт приёма-передачи животных | Да | С указанием идентификационных номеров (бирок) |
| 5 | Документы при импорте (4-8) | Если импорт | См. выше |

**Валидация:**
- Возраст тёлок: 6-18 месяцев на дату продажи в племенном свидетельстве
- Возраст нетелей: 13-26 месяцев
- Возраст баранов: 4-18 месяцев
- Возраст маточных овец: 4-18 месяцев
- При импорте: возраст на дату постановки на карантинирование у продавца

### Форма 2: Услуги по искусственному осеменению КРС/овец

| # | Документ | Обязательный | Описание |

| 1 | Акт выполненных работ | Да | От племенного центра или техника-осеменатора |

| 3 | Ведомость осеменённых животных | Да | Список с идентификационными номерами |
| 4 | Лицензия/аттестат техника-осеменатора | Да | Подтверждение квалификации |

- Докрытие при ИО: 1 баран на 100 маток

## Статусы документов в системе

| Статус | Цвет | Описание |
|--------|------|----------|
| Не загружен | Серый | Документ ещё не загружен заявителем |
| Загружен | Синий | Загружен, ожидает проверки |
| На проверке | Желтый | Специалист МИО проверяет |
| Подтверждён | Зеленый | Проверен и подтверждён |
| Отклонён | Красный | Не прошел проверку, требуется повторная загрузка |
| Запрошен повторно | Оранжевый | Заявителю отправлен запрос на повторную загрузку |
