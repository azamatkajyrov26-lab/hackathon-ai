# Структура документа "Правила субсидирования" (116 стр.)

Приказ МСХ РК №108 от 15.03.2019, ред. 2023-2025

## Основной текст (стр. 1-18)

### Глава 1. Общие положения (стр. 2-5)
- Понятия и определения
- Ключевые системы: ГИСС, ИСС, ИАС РСЖ, ЕАСУ, ИС ИСЖ, ЕГКН, ИС ЭСФ
- Субсидии — из местного бюджета, безвозмездные и невозвратные

### Глава 2. Порядок формирования и распределения субсидий (стр. 6-8)
- МИО формирует объёмы → Министерство согласовывает (2 раб. дня)
- Формула: кол-во единиц x норматив = объём субсидий
- Перераспределение между видами при нехватке заявок

### Глава 3. Условия получения субсидий (стр. 8-15)
- Кто получает: физ/юр лица, племенные центры, техники-осеменаторы
- Регистрация в ГИСС через ЭЦП
- Данные для регистрации: ИИН/БИН, ФИО, контакты, реквизиты банка
- **16 форм заявок** (Приложение 3) по разным направлениям
- Встречные обязательства (п.14-1): рост валовой продукции
- Порог: >= 100 млн тенге субсидий → обязательства обязательны
- Невыполнение 2 года подряд = блокировка на 1-2 года
- Специальная комиссия проверяет производственную мощность (раз в 3 года)

### Глава 4. Порядок приёма, рассмотрения заявок и выплаты (стр. 15-18)
- Приём заявок НЕЗАВИСИМО от наличия бюджета
- МИО проверяет за 2 раб. дня → одобрение → лист ожидания
- **ТЕКУЩИЙ ПОРЯДОК: выплата по очерёдности даты/времени регистрации (FIFO)**
- Частичная выплата: предложение → 2 дня на ответ → при отказе — следующий
- Мониторинг сохранности поголовья — ежеквартально РОСХ
- Нарушение → возврат субсидий за каждую несохранённую голову (90 раб. дней)

### Глава 5. Порядок обжалования (стр. 18)
- Жалоба руководителю МИО: 5 раб. дней
- Уполномоченный орган: 15 раб. дней
