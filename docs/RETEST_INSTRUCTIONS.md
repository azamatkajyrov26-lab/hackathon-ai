На странице **"Мое хозяйство"** (`/my-farm/`) во вкладке "Земля" появились 2 новых блока:
- **Нагрузка на пастбища** (Приказ №3-3/332) — площадь, га/голову
- **Падёж скота** (Приказ №3-3/1061) — % падежа по видам, цветовая индикация

В **детальной заявке** → вкладка "Проверки" теперь **18 фильтров** (было 15):
- Фильтр 16: Падёж скота в пределах нормы
- Фильтр 17: Нагрузка на пастбища
- Фильтр 18: Племенное свидетельство
