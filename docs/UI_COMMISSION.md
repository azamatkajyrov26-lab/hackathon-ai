# UI: Экран комиссии

**URL:** `/commission/`
**Доступ:** Комиссия (commission_member), Руководитель МИО (mio_head)

---

## Назначение

Основной рабочий экран для членов комиссии по субсидированию. Здесь комиссия:
- Просматривает ранжированный шортлист заявок
- Видит рекомендации AI с объяснениями
- Принимает решения: одобрить / отклонить / доп. проверка
- Контролирует бюджет

## Секция 4: Быстрые массовые действия

**Кнопки:**
| Кнопка | Действие | Описание |
|--------|----------|----------|
| "Одобрить все >= 70 в бюджете" | Массовое одобрение | Одобряет все заявки с баллом >= 70 в пределах бюджета |
| "Отклонить все < 40" | Массовый отказ | Отклоняет все с баллом < 40 |
| "Выбрать вручную" | Checkbox режим | Включает чекбоксы для ручного выбора |
| "Экспорт шортлиста" | Excel export | Выгрузка таблицы |

**При массовом одобрении — модальное окно подтверждения:**
```
Вы собираетесь одобрить 31 заявку на общую сумму 108,500,000 тг.
Это составляет 98.6% от остатка бюджета (110,000,000 тг).

Остаток после одобрения: 1,500,000 тг

[Подтвердить] [Отмена]
```

## Секция 5: Панель рассмотрения (при клике "Рассмотреть")

Slide-over панель справа (или отдельная страница):

### Верхняя часть: Краткая информация
```
Заявка #01300100258072  |  Балл: 94/100  |  Рекомендация: ОДОБРИТЬ
ТОО "Агрофирма Астана"  |  Акмолинская обл.  |  5,200,000 тг
```

### Средняя часть: Ключевые факторы (компактно)
```
✓ История: 18/20  — 5 субсидий, 100% выполнено
✓ Рост: 16/20  — +7.14% продукции
✓ Хозяйство: 14/15  — 230 га, 150 голов
✓ Эффективность: 13/15  — сохранность 98%
✓ Норматив: 10/10  — точное соответствие
✓ Регион: 8/10  — приоритетный
✓ Тип: 4/5  — юрлицо
✓ Первичность: 3/5  — повторный
```

### Нижняя часть: Решение

**Radio buttons:**
- ( ) Одобрить (сумма: 5,200,000 тг)
- ( ) Частично одобрить (ввести сумму: ________)
- ( ) Отклонить
- ( ) На дополнительную проверку

**Обоснование:** (textarea, обязательно при отклонении)

**При отклонении — основания (checkbox из Приложения 7):**
- [ ] Несоответствие критериям Приложения 2
- [ ] Неполнота данных или документов
- [ ] Недостоверность представленных сведений
- [ ] Несоответствие возраста/параметров животных
- [ ] Повторная заявка на уже просубсидированное поголовье
- [ ] Невыполненные обязательства по предыдущим субсидиям
- [ ] Невозврат субсидий за несохранённое поголовье
- [ ] Отсутствие регистрации в требуемых ИС
- [ ] Отсутствие земельного участка
- [ ] Нарушение встречных обязательств

**Кнопки:** "Сохранить решение" → "Следующая заявка"

Ссылка: "Открыть полную карточку" → `/applications/{id}/`

## Секция 6: Уведомления (Приложение 6 и 7)

После принятия решения система автоматически формирует:

### При одобрении — Уведомление о результатах (Приложение 6):
```
УВЕДОМЛЕНИЕ
о результатах рассмотрения заявки на субсидирование

Заявка №01300100258072 от 21.01.2025
Заявитель: ТОО "Агрофирма Астана" (БИН: 123456789012)
Направление: Приобретение маточного поголовья КРС мясного направления
Количество: 20 голов
Сумма субсидии: 5,200,000 тенге

Решение: ОДОБРЕНО
Дата решения: 27.01.2025
Выплата будет произведена в течение 2 рабочих дней.

Комиссия по субсидированию
ГУ "Управление сельского хозяйства Акмолинской области"
```

```
МОТИВИРОВАННЫЙ ОТКАЗ
в предоставлении субсидии

Заявка №... от ...
Заявитель: ...

Основание отказа:
1. Несоответствие критериям Приложения 2
2. Невыполненные обязательства по предыдущим субсидиям

Дополнительно: Заявитель не выполнил обязательства по субсидии 2023 года
(невозврат за 3 несохранённых головы).

Порядок обжалования:
- Жалоба руководителю МИО: в течение 5 рабочих дней
- Уполномоченный орган: в течение 15 рабочих дней

Комиссия по субсидированию
ГУ "Управление сельского хозяйства..."
```
