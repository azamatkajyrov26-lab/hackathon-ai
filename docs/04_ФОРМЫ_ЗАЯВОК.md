## Общие поля во всех формах заявок

1. **Данные заявителя:**
   - ИИН / БИН
   - ФИО / Наименование юрлица
   - Контактные данные (телефон, email, почтовый адрес)
   - Реквизиты банковского счёта

2. **Данные о хозяйстве:**
   - Учётный номер в ИАС "РСЖ"
   - Область / Район
   - Земельные участки (кадастровый номер из ЕГКН)
   - Членство в СПК (если применимо)

3. **Данные о животных/продукции:**
   - Направление субсидирования
   - Количество (голов / кг / доз / штук)
   - Порода / вид
   - Возраст (для животных)

4. **Финансовые данные:**
   - Норматив субсидии
   - Причитающаяся сумма (кол-во x норматив)
   - ЭСФ или таможенная декларация (номер, сумма)

5. **Подтверждение:**
   - ЭЦП заявителя
   - Дата подачи
   - Скан-копии документов

## Приложенные документы (скан-копии)

- Договор купли-продажи
- Документ, подтверждающий полную оплату стоимости
- Копия договора купли-продажи (при импорте)
- Акт карантинирования у продавца (при импорте)
- Таможенная декларация или заявление о ввозе (при импорте из-за рубежа)
- Заключение специальной комиссии (для производственных направлений)
- Документ подтверждающий членство в СПК (если кооператив)

### Приложение 4 — Перечень требований к госуслуге
- Наименование услуги
- Услугодатель (МИО)
- Способ предоставления: через ГИСС
- Срок оказания: 2 рабочих дня (проверка) + лист ожидания + 2 дня (выплата)
- Результат: уведомление о результатах или мотивированный отказ

### Приложение 4-1 — Встречные обязательства
- Форма 1: объём валовой продукции за предыдущий год
- Форма 2: реестр встречных обязательств (общедоступный)
