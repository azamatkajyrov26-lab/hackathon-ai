## Секция 1: Общий балл

**Визуал:** Большой круговой gauge (donut chart)

```
       ╭─────────╮
      │   94    │
      │  / 100  │
       ╰─────────╯
    Рекомендация: ОДОБРИТЬ
```

Цвет gauge:
- 70-100: зеленый
- 40-69: желтый
- 0-39: красный

**Текст под gauge:**
> "Высокий балл по всем факторам. Заявитель с безупречной историей, стабильный рост продукции, крупное хозяйство с высокой сохранностью поголовья. Рекомендовано к одобрению."

**Заголовок:** Мотивированный отказ по заявке №01300100258072

**Основание отказа** (checkbox, можно несколько):
- [ ] Несоответствие критериям Приложения 2
- [ ] Неполнота данных или документов
- [ ] Недостоверность представленных сведений
- [ ] Несоответствие возраста/параметров животных
- [ ] Повторная заявка на уже просубсидированное поголовье
- [ ] Невыполненные обязательства по предыдущим субсидиям
- [ ] Невозврат субсидий за несохранённое поголовье
- [ ] Отсутствие регистрации в требуемых ИС
- [ ] Отсутствие земельного участка
- [ ] Нарушение встречных обязательств (блокировка)
