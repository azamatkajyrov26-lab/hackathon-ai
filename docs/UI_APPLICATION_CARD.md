Бейдж статуса с цветом:
- Черновик — серый
- Подана — синий
- На проверке — желтый
- Одобрена — зеленый
- Отказ — красный
- Лист ожидания — оранжевый
- Выплачена — зеленый (с иконкой)

## Действия на карточке (зависят от роли и статуса)

### Заявитель:
- "Редактировать" (если status = draft)
- "Загрузить документ" (если запрошен)
- "Отозвать заявку" (если status = submitted)

### Специалист МИО:
- "Проверить документы"
- "Запросить доп. документы"
- "Запустить проверку" (re-run hard filters)
- "Отклонить" → форма мотивированного отказа

### Комиссия:
- "Одобрить" → подтверждение суммы
- "Частично одобрить" → ввод суммы
- "Отклонить" → выбор основания отказа + комментарий
- "На доп. проверку" → комментарий

### Руководитель МИО:
- "Утвердить решение комиссии"
- Все действия комиссии
