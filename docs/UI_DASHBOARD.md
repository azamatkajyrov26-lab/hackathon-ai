| # | Балл | Рекомендация | Заявитель | Направление | Район | Сумма | Действие |
|---|------|-------------|-----------|-------------|-------|-------|----------|
| 1 | 94 | Одобрить | ТОО "Агрофирма Астана" | Маточное КРС | Целиноградский | 5,200,000 | Рассмотреть |

- Зеленый (70-100): рекомендовано одобрить
- Желтый (40-69): на усмотрение комиссии
- Красный (0-39): рекомендован отказ

## Фильтры (HTMX — без перезагрузки)

Панель фильтров над таблицей:

| Фильтр | Тип | Значения |
|--------|-----|----------|
| Область | select | 17 областей РК |
| Район | select (зависимый от области) | Районы выбранной области |
| Направление | select | Мясное, Молочное, Овцеводство, Птицеводство, Коневодство, Свиноводство |
| Статус | multiselect | Подана, Проверка, Одобрена, Отказ, Лист ожидания, Выплачена |
| Балл от | number input | 0-100 |
| Балл до | number input | 0-100 |
| Дата от | date picker | - |
| Дата до | date picker | - |
| Поиск | text input | По номеру заявки, ИИН/БИН, имени |

Фильтры обновляют таблицу через HTMX без перезагрузки страницы.
