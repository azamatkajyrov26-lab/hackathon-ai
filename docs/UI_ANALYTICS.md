## Секция 2: Графики (Chart.js)

### График 1: Распределение баллов (Histogram)
**Тип:** Bar chart (вертикальный)
**Ось X:** Диапазоны баллов (0-10, 10-20, ..., 90-100)
**Ось Y:** Количество заявок
**Цвет:** Красный (0-39), Желтый (40-69), Зеленый (70-100)

### График 2: Заявки по регионам (Horizontal Bar)
**Тип:** Horizontal bar chart
**Ось Y:** 17 областей
**Ось X:** Количество заявок
**Цвет баров:** градиент по среднему баллу региона

Пример данных:
| Область | Заявок | Средний балл | Сумма |
|---------|--------|-------------|-------|
| Алматинская | 189 | 74.2 | 520M |
| Акмолинская | 156 | 71.8 | 450M |
| Костанайская | 134 | 69.5 | 380M |
| Туркестанская | 112 | 65.3 | 290M |
| Карагандинская | 98 | 72.1 | 340M |
| область Абай | 87 | 68.7 | 240M |
| ... | | | |

### График 3: Заявки по направлениям (Pie / Donut)
**Тип:** Donut chart
**Сегменты:**
| Направление | % | Сумма |
|------------|---|-------|
| Мясное скотоводство | 34% | 850M тг |
| Молочное скотоводство | 22% | 550M тг |
| Овцеводство | 18% | 450M тг |
| Птицеводство | 10% | 250M тг |
| Коневодство | 8% | 200M тг |
| Свиноводство | 5% | 125M тг |
| Прочие | 3% | 75M тг |

### График 4: Динамика по месяцам (Line chart)
**Тип:** Multi-line chart
**Ось X:** Месяцы (январь — декабрь)
**Линии:**
- Поданные заявки (синий)
- Одобренные (зеленый)
- Отказанные (красный)
- Средний балл (правая ось Y, пунктир)

### График 5: Бюджет план/факт (Stacked Bar)
**Тип:** Stacked bar по регионам
**Сегменты:** Освоено (зеленый) + Остаток (серый)
**Линия:** Плановый бюджет (пунктир)

### График 6: Топ-10 заявителей по сумме субсидий (Bar chart)
**Тип:** Horizontal bar
**Данные:** Имя заявителя → совокупная сумма полученных субсидий
