## Шаг 7: Загрузка документов

Динамический список документов — зависит от вида субсидии и наличия импорта:

### Обязательные для ВСЕХ направлений:
| Документ | Формат | Макс. размер |
|----------|--------|-------------|
| Договор купли-продажи | PDF, JPG, PNG | 10 MB |
| Подтверждение оплаты стоимости | PDF, JPG, PNG | 10 MB |
| ЭСФ (электронная счет-фактура) | PDF | 10 MB |

### Дополнительные при ИМПОРТЕ:
| Документ | Формат | Когда |
|----------|--------|-------|
| Копия договора купли-продажи (на языке оригинала + перевод) | PDF | Импорт |
| Акт карантинирования у продавца (экспортёра) | PDF | Импорт |
| Таможенная декларация | PDF | Импорт из дальнего зарубежья |
| Заявление о ввозе товаров | PDF | Импорт из ЕАЭС |

### Дополнительные по направлению:
| Документ | Формат | Когда |
|----------|--------|-------|
| Заключение специальной комиссии (Приложение 5) | PDF | Формы 5, 6, 7, 8 (производственные) |
| Документ о членстве в СПК | PDF | Кооператив (Форма 7) |
| Племенное свидетельство | PDF | Формы 1, 13 (племенное поголовье) |
| Ветеринарный сертификат | PDF | Формы 1, 9, 10, 11, 12 |
| Акт взвешивания | PDF | Формы 9, 10, 11, 12 (откорм/убой) |

**UI:** Drag & drop зона для каждого документа. Статус: "Загружен" / "Не загружен" / "На проверке".

## Шаг 8: Подтверждение и отправка

**Сводка заявки (read-only):**
```
Направление:      Мясное скотоводство
Вид субсидии:     Приобретение маточного поголовья КРС — отечественный
Заявитель:        ТОО "Агрофирма Астана" (БИН: 123456789012)
Регион:           Акмолинская область, Целиноградский район
Количество:       20 голов
Норматив:         260,000 тг/голова
Сумма:            5,200,000 тг
ЭСФ:              ESF-2025-0001234 на 7,800,000 тг
Продавец:         ТОО "Племенное хозяйство Алтай" (БИН: 987654321098)
Документы:        4 из 4 загружены
```

**Checkbox обязательный:**
- [ ] Подтверждаю достоверность представленных сведений и даю согласие на проверку данных в государственных информационных системах

**Кнопки:**
- "Сохранить черновик" → status = draft
- "Подписать ЭЦП и отправить" → status = submitted (в MVP — просто кнопка без реальной ЭЦП)

## Валидация

### На клиенте (Alpine.js):
- ИИН/БИН: ровно 12 цифр
- Банковский счет: начинается с KZ, 20 символов
- Возраст животных: в допустимом диапазоне для категории
- Сумма: количество × норматив
- Все обязательные поля заполнены
- Все обязательные документы загружены

### На сервере (Django):
- ИИН/БИН существует в системе
- Животные верифицированы через ИС ИСЖ
- ЭСФ подтверждена через ИС ЭСФ
- Земельный участок подтвержден через ЕГКН
- Нет дубликатов заявок на те же животные
